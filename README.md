# Zoom Integration Concept App - Simone Zappino

Integrazione SDK Zoom per l'avvio di videochiamate all'interno di una pagina Web.
Il Concept è stato sviluppato utilizzando [Vue](https://cli.vuejs.org/ "Vue") a scopo dimostrativo, l'SDK può essere implementato in qualunque framework front-end oltre che in plain Javascript.

L'app attualmente non supporta la creazione e la gestione di videochiamate (implementabile tramite le API di zoom).

## Installazione dependencies
```
npm install
```

### Compilazione e hot-reload ambiente sviluppo
```
npm run serve
```

### Compilazione ambiente produzione
```
npm run build
```
